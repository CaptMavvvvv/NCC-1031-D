# Python Set - กลุ่มข้อมูลที่ไม่ซ้ำและไม่มีลำดับ


#### Set Characteristic
> สร้างด้วยวงเล็บปีกกา { } (ใส่ข้อมูลดิบลงไปตรง ๆ)
> ไม่มีลำดับ (Unordered): ข้อมูลจะกระจายตัวอยู่ ไม่มี Index อ้างอิง ตำแหน่งเปลี่ยนได้ตลอดเวลา
> ห้ามมีข้อมูลซ้ำ (No Duplicates): ถ้าใส่ข้อมูลซ้ำลงไป มันจะยุบเหลือตัวเดียวโดยอัตโนมัติ
> แก้ไขได้ (Mutable): สามารถเพิ่มหรือลบสมาชิกได้ (แต่ตัวข้อมูลข้างในต้องเป็นประเภทที่เปลี่ยนค่าไม่ได้)
> เหมาะสำหรับ: การทำหาความสัมพันธ์ทางคณิตศาสตร์ (เช่น หาตัวที่ซ้ำกัน Union, Intersect) หรือใช้กำจัดข้อมูลที่ซ้ำทิ้งไป
